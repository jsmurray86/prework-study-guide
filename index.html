  • The head element contains information about the webpage. It also contains the meta data or data about the website.
  • The body element contains the the visible content that the user sees.
  • HTML elements are case sensitive.
  • When using headings, make sure they're in order. Use h1 for example, before h2 for users who use screen readers or other accessibility tools.
  • alt attribute contains a string that describes the appearance and functionality of the image. This is also important for screen readers and other adaptive technologies.
  • Semantic HTML elements indicate the role of their content as well as giving structure to the content of the webpage.
